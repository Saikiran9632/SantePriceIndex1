# Sante-Price Index

Sante-Price Index is an Android application designed to help users track prices and calculate profit margins effectively.

## Features

- **Price Watch**: Track and monitor prices for various items.
- **Profit Calculator**: Calculate recommended retail prices based on purchase price, transport costs, and expected profit.
- **Digital Price Board**: A digital display for price information.

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: XML Layouts with Material Design 3
- **Build System**: Gradle (Kotlin DSL)
- **Minimum SDK**: 24 (Android 7.0)
- **Target SDK**: 35 (Android 15)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/SantePriceIndex.git
   ```
2. Open the project in Android Studio.
3. Sync Gradle and run the app on an emulator or physical device.
